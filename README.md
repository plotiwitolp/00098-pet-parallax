# Pet Parallax

Интерактивная параллакс-анимация с милыми животными, созданная с использованием чистого CSS и JavaScript.

## Описание

Этот проект представляет собой интерактивную веб-страницу с эффектом параллакса, где слои изображений с животными перемещаются с разной скоростью при движении курсора мыши или прокрутке страницы. Проект демонстрирует возможности современных CSS-анимаций и JavaScript для создания захватывающих визуальных эффектов.
